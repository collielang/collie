- Date: 日期
- Time: 时间
- DateTime: 日期时间（Timestamp）
- TimeSpan: 时间差

以及它们之间的互相转换

- DateFormat: 日期字符串格式化
- TimeZone: 时区

---

更多类型：

- Month: 月份
- Season: 季度
- Week: 星期
